# 📊 PocketPlanner

PocketPlanner is a personal budget tracking website built with HTML, CSS, JavaScript (Frontend) and Node.js, Express, SQLite3 (Backend). It helps users manage income and expenses efficiently.

## 🚀 Features

- Add, edit, delete transactions
- Filter and view transaction summary
- User login and registration
- Responsive multi-page layout

## 💻 Tech Stack

- HTML, CSS, JavaScript
- Node.js, Express.js
- SQLite3 with Knex.js

## ▶️ Run Locally

```bash
npm install
npm start
